# ts3ctl
An automatic Teamspeak 3 update script.

## USAGE
```bash
./ts3ctl -i
```
Installs the latest ts3server.

```bash
./ts3ctl -c
```
Checks for updates.

```bash
./ts3ctl -s
```
Updates ts3server silently.

```bash
./ts3ctl -u
```
Updates ts3server.

```bash
./ts3ctl -v
```
Version and releaseinfo.

```bash
./ts3ctl -h
```
Shows the help.
