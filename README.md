# ts3_update
An automatic Teamspeak 3 update script.

## USAGE
```bash
./ts3_update -c
```
Checks for updates.

```bash
./ts3_update -s
```
Updates ts3server silently.

```bash
./ts3_update -u
```
Updates ts3server.

```bash
./ts3_update -v
```
Version and releaseinfo.

```bash
./ts3_update -h
```
Shows the help.
